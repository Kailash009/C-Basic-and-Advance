Dot Net Components:-
------------------------------------------
There are two main components in dot net which are as follows -
1. CLR. (Common Language Runtime)
>> All dot net applications are executed by CLR.
	CLR has some sub-components:-
		a) CTS(Common Type System)
>> All primitive data types (int,float,char,double,byte,bool) are defined into CTS.
		b) CLS (Common Language Specification)
>> you can write your code with any language,it will not make any difference.
		c)  GC(Garbage Collector)
>> It is responsible to release/deallocated your occupied memory by the program.
-----------------------------------------------------------------------------------------------------------------------------------------
		d)  Native Code.
		>> It is byte code/Machine Code that can executed with any machine.
--------------------------------------------------------------------------------------------------------------------------------------
2. BCL/FCL. (Base class Library/Framework Class Library).
>> It provide predefined methods,libraries,classes and interfaces.

Compilation process of C#:-
------------------------------------------------------------------
>> C# is a two-way compilation process.
1. In First Compilation process, our C# compiler(.csc) converted source code into MSIL Code.
2. In Second Compilation process,our clr gives one additonal compiler i.e. JIT compiler converts our MSIL Code into Native Code/Byte Code.
