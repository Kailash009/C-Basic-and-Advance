Component of Dot Net Framework:-
------------------------------------------------------------------------
There are two main components in dot which are as follows:-
1. CLR. 
>> It stands for Common Language Runtime.
>> CLR is responsible to execute all your dot net applications.
>> CLR is the heart of dot net framework.
CLR has some sub-component:-
	a) CTS (Common Type System) 
>> All primitive data types (int,char,float,bool,double) are defined into CTS.
	b) CLS (Common Language Specification)
>> After compilation of your code you didn't get any difference. (MSIL).
	c) GC (Garbage Collector)
>> GC automatically destroy you allocated objects after use it.
	d) Native Code.
>> It is a Byte code that execute in any O.S.(win,mac,linux)
-----------------------------------------------------------------------------------------------------
C# Compilation Process:-
-------------------------------------------------------
>> It is a two-way compilation process.
---------------------------------------------------------------------------------------------------------------------
2. BCL/FCL
>> It stands for Base Class Library/Framework class Library.
>> All predefine methods,classess or interfaces are defined into BCL.
In C -> #include<stdio.h>  // BCL 
printf()
scanf()
---------------------------------------------------------------------
In C#
using System;  //BCL/FCL
Console.readLine() -> to take input from the user.
Console.WriteLine() -> to display the result.

Constant
>> It is a fixed value i.e. never going to be changed through out the whole program.
There are three types-
1. Numeric 
>> it hold numeric value. 1,2,3,.......
a) Octal  -> 0 to 7.
b) Decimal -> 0 to 9.
c) Hexadecimal -> 0 to 15.
d) Real or Floating Point -> 23,  45.67 etc.
------------------------------------------------------------------------------------------------------------
2. Character. 
>> It is represented by single quotes.
>> it contains only single value.
'a'    '3'   '$' etc.
----------------------------------------------------------------------------------
3. String
>> it is represented by double quotes.
>> it contains more than one character.
For ex- 
"lfjdlskf" , "43434"  , "dfsdd3434&^%^&%&" etc.

Data type-
--------------------
>> It identifies that which type of value a variable can hold.
---------------------------------------------------------------------------------

2. Non-Primitive or Reference Data type.
