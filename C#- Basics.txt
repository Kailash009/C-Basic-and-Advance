Dot Net Components:-
--------------------------------------
There are two major components in dot net which are as follows:-
a) CLR. (Common Language Runtime)
>> All dot net applications are executed/run by CLR.
>> it is the heart of dot net framework.
b) BCL/FCL. (Base Class Library)/Framework Class Library.
>> All predefined methods,classes,interfaces,libraries are defined into BCL/FCL.
	a) CLS (Common Language Specification)
>> you can write your code with any language,after compilation of your code you didn't get any difference.
	b) CTS (Common Type System)
>> All primitive data types (int,bool,float,double,char,byte) are defined into CTS.
	c) GC (Garbage Collector)
>> it is used for automatic memory management, it release all your occupy memory which is taken by the program.
	d) Native Code.
>> It is a byte code , which is easily understandable by any computer. (machine Code)-> 101001001

Compilation Process of C#:-
--------------------------------------------
>> It is a two-way compilation process.
1. In First Compilation process our source code coverted into MSIL Code.
2. In Second Compilation process MSIL code converted into Native/Byte Code with the help of JIT (Just-in Time Compiler) Compiler.
